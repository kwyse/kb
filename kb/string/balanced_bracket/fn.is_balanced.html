pub fn is_balanced(brackets: Brackets) -> bool

Checks if a string of brackets is balanced

Only strings that are made up exclusively of ASCII brackets are allowed. It makes use of two notable optimizations. First, if the length of the input is odd, the brackets cannot be balanced, so we return false immediately. Secondly, if we ever reach a closing bracket and the internal stack holding closing brackets is empty, it means there was no corresponding opening bracket earlier in the string, so we return false immediately.

Tests

let mut brackets = Brackets::from_bytes(b"{{[]}}[][]".to_vec()).unwrap();
assert!(is_balanced(brackets));

brackets = Brackets::from_bytes(b"{{[]".to_vec()).unwrap();
assert!(!is_balanced(brackets));
